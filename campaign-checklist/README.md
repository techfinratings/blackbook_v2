# 마케팅 캠페인 SOP — 모듈형 체크리스트 생성기 v2.0

`index.html` 하나로 완결되는 오프라인 도구입니다. 사내 공유 폴더에 이 파일만 복사해 배포하면 됩니다.

- 외부 네트워크 요청 0건 (CDN·fetch·폰트 로드 없음), 브라우저 스토리지 미사용
- Excel(.xlsx)은 라이브러리 없이 OOXML을 직접 생성 — 상태 드롭다운(미착수/진행중/완료/N.A),
  상태별 조건부 서식, A4 세로 1페이지 폭(fitToWidth), 인쇄 영역·헤더 행 반복 포함

## 항목 수정 방법 (비개발자용)

`index.html`을 메모장으로 열고 `[데이터 정의]` 주석 블록(파일 중간, `<script>` 시작 직후)의
배열 문자열만 고치면 됩니다.

| 수정 대상 | 상수 |
|---|---|
| 모듈 목록 | `MODULES` |
| 모듈별 과업 행 | `TASKS` — `[NO, 과업, 체크리스트, 협업부서, 비고]` |
| **노출 채널 목록 (미확정 초안)** | `CHANNELS` — 채널 추가/삭제/이름 변경은 이 배열만 갱신 |
| 채널 공통 5단계 과업 세트 | `CHANNEL_TASK_SET` |
| 검수 자동 구성 규칙 | `INSPECTIONS` |
| 종속 규칙 R1~R7 | `DEP_RULES` (R8·R9는 화면 로직) |
| 프리셋 A~E | `PRESETS` |
| 단계별 기한(D-) 기본값 | `STAGE_DDAY` |

수정 후 브라우저에서 열어 프리셋 A를 눌러 미리보기·Excel 다운로드가 정상인지 확인하세요.
